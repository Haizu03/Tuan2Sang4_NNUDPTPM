# Bài 2 Tuần 2 – JSON Server Demo

## 1. Mục tiêu
- Sử dụng json-server để tạo REST API giả lập
- Thực hiện CRUD với posts và comments
- Áp dụng xoá mềm cho posts
- Tự động tăng ID khi tạo mới

## 2. Công nghệ sử dụng
- HTML, JavaScript
- json-server
- Node.js

## 3. Cách chạy project

npm install json-server
